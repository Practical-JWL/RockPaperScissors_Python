## ✊✌️✋ 5전 3선승제 가위바위보 게임
Python Flet 라이브러리를 활용하여 제작한 GUI 기반의 가위바위보 게임임.

- 사용자와 컴퓨터가 대결하여 먼저 3승을 달성하는 쪽이 최종 우승하는 5전 3선승제 시스템을 구현함.

### 1. ✨ 주요 기능
- 5전 3선승제 로직: nonlocal 변수를 사용하여 승점을 실시간으로 관리하고, 누군가 3승에 도달하면 게임이 자동 종료됨.

- 이벤트 핸들링: lambda 함수를 활용해 하나의 함수(play_game)로 가위, 바위, 보 입력을 효율적으로 처리함.

- 상태 제어: 게임 종료 시 버튼을 비활성화(disabled = True)하여 추가 클릭을 방지하는 방어적 설계를 적용함.

---

### 2. 🧠 논리 전개 과정
- 데이터 설계: 승점(user_wins, computer_wins)과 선택지 리스트를 초기화함.

- UI 배치: 점수판, 버튼 열(Row), 결과창을 순서대로 배치함.

- 승패 판정: 사용자의 선택과 컴퓨터의 랜덤 선택을 비교하여 3가지 케이스(승/패/비김)로 분기 처리함.

- 결과 반영: page.update()를 호출하여 스코어와 메시지를 즉각 갱신함.

---

### 3. 🚨 트러블슈팅: 레이아웃 고정 (Stable Layout)
❌ 문제 상황
현상: 브라우저 화면을 확대하거나 창 크기를 조절할 때, 중앙에 있던 가위/바위/보 버튼들이 창의 너비에 따라 좌우로 움직이는 현상 발생.

원인: flet.Row의 alignment=CENTER 속성은 가용한 전체 가로 길이를 기준으로 중앙을 계산하기 때문에, 창 크기가 변하면 '상대적 중앙값'이 계속 바뀌게 됨.

✅ 문제 해결
페이지의 축(Axis) 정렬 방식을 명시적으로 지정하여 UI 요소들을 화면 정중앙에 고정.

```
# 수정 전: 기본 정렬 (창 크기에 따라 요소가 유동적임)

# 수정 후: 축 정렬 고정
page.horizontal_alignment = flet.CrossAxisAlignment.CENTER # 가로 중앙 고정
page.vertical_alignment = flet.MainAxisAlignment.CENTER    # 세로 중앙 고정
```
결과: 창 크기를 조절하거나 화면을 확대해도 모든 UI 요소가 흩어지지 않고 지정한 위치에 안정적으로 고정되어 사용자에게 일관된 인터페이스를 제공함.
